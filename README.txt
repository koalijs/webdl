Webdl
=====================

Web page resource sniff and crawl through puppeteer

Usage: webdl open [options] <url> [dest]

Open web page for track request files

Options:
  -f, --filter <text>  filter urls for save by the adblock filter ruler. see: https://adblockplus.org/filter-cheatsheet
  -s, --slient         Don't show logs
  -o, --override       Override files
  -h, --help           display help for command
